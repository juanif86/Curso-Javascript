El proyecto incluye los siguientes Elementos e interacciones con Javascript:

. Control de flujo:
        - Uso de Switch;

. Objetos y Constructor de Objetos:
	- Las compras son un objeto;
	- Los productos son otro objeto;

. Arrays:
	- Array para guardar productos;
	- Array para guardar compras;

. Funciones:
        - function pintaCard(array);
        - function almacenaEnLS();
        - function getData(e);
        - function listarCompra(cant, nom);
        - function calcularCantidad(monto, precio);

. Higher Order Functions:
        - Uso de operador ternario;

. Uso de DOM: 
      - Saludo al loguearse; 
      - Renderizado de Cards;
      - Captura de datos de form;
      - Creación de Divs desde JS;
      - Asignacion de clases desde JS;

. AddEventListener:
        - Click;

. Uso de LocalStorage: 
      -LogIn/LogOut;

. Librerías:
	- SweetAlert;
        - Particles;
. JSON:
 	- Uso de archivo JSON con propiedades de librería Particles;

. Promesas:
        - uso de Then;

. Fetch:
        - traigo datos de API;
